# CSR Chatbot (UR Edition)

A simplified Customer Service Representative chatbot that books technician appointments and answers basic FAQs about services and coverage areas.

> 🚧 This README is a stub and will be filled out as the project progresses. See `PRD.md` for the living build plan.

## Quickstart

```bash
# Create and activate virtual environment
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate

# Install dependencies
pip install -r requirements.txt

# Run tests
pytest

# Run CLI (available after Phase 5)
python -m src.cli
```
